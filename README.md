# Workshop #4: Safety-Critical Robotics

Static website project for the workshop:

**Safety-Critical Robotics: From Design Principles to Safe Autonomy in Industrial and Human-Centered Environments**

## Files

- `index.html` - page content and structure
- `styles.css` - responsive visual design
- `main.js` - navigation and section highlighting
- `assets/safety-critical-robotics-hero.png` - generated hero image

## Preview

Open `index.html` directly in a browser. No build step is required.
